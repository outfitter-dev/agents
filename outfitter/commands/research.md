# Technical Research

Conduct thorough research on the given topic using the systematic research methodology.

## The Process

1. **Question Framing** - Clarify scope, identify key terms, define success criteria
2. **Discovery** - Search official docs, standards, benchmarks, case studies
3. **Evaluation** - Assess source credibility, cross-reference claims
4. **Comparison** - Build decision matrices when comparing options
5. **Recommendation** - Synthesize findings with confidence levels and citations

## Source Hierarchy

1. Official documentation and specifications
2. Standards bodies and RFCs
3. Peer-reviewed benchmarks
4. Production case studies
5. Reputable community sources
